	Используемый алгоритм:
По нажатию Enter сохраняется исходное изображение, являющееся фоном.
Далее каждый кадр сравнивается с исходным и определяется наличие новых объектов.
Координаты этих объектов сравниваются с соответствующими координатами со следующего кадра (используется алгоритм для определения оптического потока).
В случае пересечения объектом требуемой линии, в данном случае разделяющей кадр на левую и правую часть, происходит учет такого пересечения.
Для увеличения скорости работы программы используется параллельная обработка нескольких частей кадра (использован Thread Pool).
